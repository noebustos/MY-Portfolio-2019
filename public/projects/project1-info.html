Netflix HTML email

I made this project from emails that shows up on my promotions tab. It's an exact clone of the "comming soon" emails that Netflix sends out to their subscribers. Every line of code was typed by me and uses HTML email best practices. The design was coded to look the same as the original Netflix email.

Inline CSS was done to make sure that email clients would have no problem converting code into a visual email. Along with inline CSS, a style tag with media querries as children, is also used for email clients that do offer mobile compatibility. The HTML body has tables nested with more tables, and thus ensurres more compatibility with multiple email clients. The final step after completing the HTML email was to test it by sending it to an actuall email address. this was done multiple times to make sure everything is as desired and no bugs cause things to break and not display to the email client.

technologies implemented

PutsMail / Litmus

Putsmail.com was used for testing and debuging. This site allows you to send your HTML email to multiple email addresses. Litmus.com allows you to see a preview of what the end product looks like on multiple email providers. Using this two tools, our HTML email is ready to send to an email address list, and it's also mobile compatible.

Image compression

One of our last step, before our email is ready to send, is to optomise our images. Using sites like tinypng, our images are compressed and reduced in file size to make our html email load faster and shrink in file size.
